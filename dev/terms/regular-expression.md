# 정규표현식(정규식) regular expression

egular expression (약어: regex, regexp)

프로그래밍에서 문자열을 다룰 때, 문자열의 일정한 패턴을 표현하는 형식

ex. ^\[0-9]\*$ : 숫자, ^\[a-zA-Z]\*$ : 영문자
