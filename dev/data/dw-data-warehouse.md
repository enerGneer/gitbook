# DW (Data Warehouse)

* 사용자의 의사 결정에 도움을 주기 위하여, 기간시스템의 데이터베이스에 축적된 데이터를 공통의 형식으로 변환해서 관리하는 데이터베이스.
* 분산되어있는 각각의 데이터베이스 관리 시스템들을 통합하여 관리한다.
* 중앙 저장소 역할을 한다.
